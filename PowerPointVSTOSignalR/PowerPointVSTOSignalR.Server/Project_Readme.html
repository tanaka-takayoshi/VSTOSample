ユーザーの ASP.NET アプリケーション

プロジェクトの 作成が完了しました

このアプリケーションの構成:

  • [Home]、[About]、[Contact] への基本的なナビゲーションを示したサンプル ページ
  • Bootstrap を使用したテーマ
  • 認証を選択した場合、登録とサインインの方法を表示
  • NuGet を使用して管理する ASP.NET 機能

アプリケーションのカスタマイズ

  • ASP.NET MVC を使い始める
  • サイトのテーマを変更する
  • NuGet を使用してライブラリを追加する
  • 認証を構成する
  • Web サイト ユーザーの情報をカスタマイズする
  • ソーシャル プロバイダーから情報を取得する
  • ASP.NET Web API を使用して HTTP サービスを追加する
  • Web API をセキュリティで保護する
  • ASP.NET SignalR でリアルタイム Web を追加する
  • Scaffolding を使用してコンポーネントを追加する
  • Browser Link でアプリケーションをテストする
  • プロジェクトを共有する
